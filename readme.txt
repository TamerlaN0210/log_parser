Для работы приложения требуется:
1. Интерпретатор python 3.6
2. БД MySQL
3. Сервер для отображения веб-страницы

Для работы парсера с БД нужны библиотека mysql-connector-python 8.0.13
pip install mysql-connector-python
И библиотека Requests
pip install requests

В папке web хранятся страницы для сервера
Перед запуском веб-страницы нужно указать данные для доступа к БД, которые находятся в  файле settings.php
и адрес сайта в questions.js

Для запуска работы парсера нужно запустить файл main_prog.py и следовать указаниям.

Если IP перестали определятся онлайн, возможно, вас забанило на сайте http://ip-api.com
Для разбана перейдите на http://ip-api.com/docs/unban и укажите свой IP адрес.
