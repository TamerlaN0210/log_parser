Для работы приложения требуется:
1. Интерпретатор python 3.6
2. БД MySQL
3. Сервер для отображения веб-страницы

Для работы парсера с БД нужна библиотека mysql-connector-python 8.0.13, установка через PIP
pip install mysql-connector-python

Перед запуском вед-страницы нужно указать данные для доступа к БД, которые находятся в  файле settings.php

Для запуска работы парсера нужно запустить файл main_prog.py и следовать указаниям.

Если IP перестали определятся онлайн, возможно, вас забанило на сайте http://ip-api.com
Для разбана перейдите на http://ip-api.com/docs/unban и укажите свой IP адресс.
